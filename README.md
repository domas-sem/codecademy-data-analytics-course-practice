# CodeAcademy Data Analytics Course Practice

This repository contains my Python and SQL practice tasks completed during the CodeAcademy Data Analytics course.

Its purpose is to document my learning process, organize my practice work, and support my junior data analyst portfolio.

## Repository structure

- `python/` – Python practice tasks and mini projects
- `sql/` – SQL practice queries and mini projects

## Featured mini projects

### Python: Medical insurance dictionary project

A beginner Python project focused on working with dictionaries, nested dictionaries, loops, dictionary comprehensions, and simple summary calculations.

Main skills demonstrated:

- dictionary creation and updates
- `zip()` and dictionary comprehension
- nested dictionary structures
- loops and formatted output
- working with small structured datasets

File:

- `python/medical_insurance_dictionary_project.py`

### SQL: Sakila SQL analysis

A SQL practice project based on the `sakila.film` table from the Sakila sample database.

Main skills demonstrated:

- filtering with `WHERE`
- aggregation with `SUM`, `COUNT`, `AVG`, `MIN`, `MAX`
- grouping with `GROUP BY`
- sorting with `ORDER BY`
- pattern matching with `LIKE`
- conditional logic with `AND`, `OR`, and `BETWEEN`

File:

- `sql/sakila_tasks.sql`

## Topics covered

- SQL queries
- Filtering and sorting
- Aggregations
- Joins and multi-table queries
