# SQL tasks

This folder will contain SQL tasks and practice queries from my CodeAcademy courses learning.
Public databases such as Lyft Trip Data, Startups, Sakila have been used for executing these tasks.


## Files

- `aggregate_functions_task.sql` – practice with aggregate functions (SUM, COUNT, AVG, etc.).
- `multiple_tables_trip_data.sql` – joining multiple tables on trip/travel data.
- `sakila_tasks.sql` – queries on the Sakila sample database (films, rentals, ratings).
