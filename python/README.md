## Featured mini project: Medical insurance dictionary project

This Python file focuses on building and working with dictionary-based data structures using a small medical insurance dataset.

### Focus areas

- Creating and updating dictionaries
- Using `update()` and `get()`
- Calculating summary metrics such as average insurance cost
- Combining lists with `zip()`
- Building dictionaries with dictionary comprehension
- Working with nested dictionaries
- Iterating through records with `for` loops
- Printing formatted summaries from structured data

### Project overview

In this exercise, patient names, ages, and insurance-related information are stored and manipulated using Python dictionaries.

The script includes:

- a dictionary of patient insurance costs
- a calculated average insurance cost
- a dictionary mapping names to ages
- a nested dictionary of full medical records
- formatted output describing each patient record
